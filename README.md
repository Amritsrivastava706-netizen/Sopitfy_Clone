# 🎧 Spotify Clone (Pure HTML, CSS & JavaScript)

A modern, responsive **Spotify-inspired music player** built using **vanilla HTML, CSS, and JavaScript** — no frameworks used.

This project replicates core Spotify features like music playback, playlists, and interactive UI with a clean and smooth user experience.

---

## 🚀 Features

* 🎵 Play / Pause songs
* ⏭ Next / Previous track controls
* 🔀 Shuffle mode
* 🔁 Repeat mode
* ❤️ Like songs (UI interaction)
* 📊 Progress bar with seek functionality
* 🔊 Volume control + Mute toggle
* 🎧 Now Playing section
* 🎚 Keyboard shortcuts support
* 🎨 Modern Spotify-inspired UI
* 📱 Responsive design (works on smaller screens)

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Flexbox + Grid)**
* **JavaScript (Vanilla JS)**
* **Font Awesome Icons**
* **Google Fonts (DM Sans)**

## ⚙️ Setup & Usage

1. Clone the repository:

```bash
git clone https://github.com/your-username/spotify-clone.git
```

2. Navigate into the project folder:

```bash
cd spotify-clone
```

3. Add your audio files inside the `song/` folder:

```
song/1.mp3
song/2.mp3
...
```

4. Open `index.html` in your browser.

## 🎹 Keyboard Shortcuts

| Key   | Action         |
| ----- | -------------- |
| Space | Play / Pause   |
| →     | Next Song      |
| ←     | Previous Song  |
| M     | Mute / Unmute  |
| S     | Toggle Shuffle |
| R     | Toggle Repeat  |
